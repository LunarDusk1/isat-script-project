THE ISAT SCRIPT

The reason for this website's creation. I shall attempt to make a full script of hit indie game In Stars and Time, which everybody should play immediately.

All (most) images used are taken from game files and hosted on The fantabulous ISATwiki!!!!

Index

  • Story (Events specifically tied to story progression)
  • Events (Events that only occur once/under special circumstances, e.g. Ghosts event, as well as sidequests)
  • Rooms (Things tied to specific locations that can generally be repeated at will)
  • Random (Whatever: Items, dialogue repeated across rooms)
  • START AGAIN (Pages related to START AGAIN: a prologue)
  • Special Thanks (The cool people who made this possible!)

Site News

Shaking and waving text

Date of notice: 15th July, 2024

The decision's been made to depict shaking/waving text from the game as italics. We do actually have a way to pull off shaking text in html (Gold is a wizard), but I (Feli) personally do not like emulating the game that closely for stupid reasons.

So italics it is. Also it's way easier. Shaking text is possible but hot diggity.

Retroactively adding that to old pages is obviously a huge pain so don't expect it to be 100% accurate please. Because it is not and probably never will be.


New Buttons!

Date of notice: 12th July, 2024

You may notice some fancy new buttons at the bottom right of the site! These are the brand new Settings buttons!


A short list of buttons:

  • Settings: Open the remaining settings.
  • Go to top: Go back to the top of the page.
  • Toggle expressions: turn expressions (the little things in brackets) on/off.
  • Toggle choices: Open/close all dialogue choices!
  • Default font: Set font to default.
  • Cascadia Code: Set all text to Cascadia Code.
  • OpenDyslexic3: Set all text to OpenDyslexic3.
