Gaining EXP from death

Source: Common Event 062 (BATTLE_StartBattle); Troop54 (BASE TROOP)

After certain deaths, you'll gain EXP next time you enter a battle. This text plays after buffs are applied but before anyone takes a turn.

After dying to a Sadness

(A Sadness killed you earlier... But you feel a tiny bit stronger.)

(You gained a little bit of EXP[!/.])


Siffrin gains 500 EXP.

After dying to the King

(You didn't manage to beat the King earlier, but you feel a little stronger.)

(You gained some EXP[!/.])


Siffrin gains 2000 EXP.

After getting to the end

(You managed to beat the King earlier...)

(You feel stronger.)

(You gained a lot of EXP.)


Siffrin gains 10000 EXP.


Bonnie helps

Source: Common Event 048 (BONNIEACTION); Skills 052-059

Every second turn, Bonnie will try to help.

During the tutorial

Bonnie (stop!2) I WANNA HELP TOO!!!

If Memory of Bonnie, Memory of Snacks, or Memory of Promise is equipped, the game rolls a number between 1 in 3. If a 1 is rolled, Bonnie will try to take the action specified by their Memory (attacking, healing, or buffing, respectively).

If not, the game rolls a number between 1 and 5.


Bonnie will try to heal if a 1 is rolled

If Isabeau's HP is below 80%, Bonnie will heal the whole party by 20%. If not, they'll try to buff instead.


(Bonnie runs in with snacks!)


Bonnie will try to heal if a 2 is rolled, or if they couldn't heal

If Bonnie has the Griddle Pan, the Wok, or the unused Saucepan equipped, they'll always try to buff ATK, DEF, or ATKSPD, respectively.

If not, Bonnie will buff anything Siffrin is debuffed in, with the priority of ATKSPD over ATK over DEF.

If not, a buff will be picked at random. If Siffrin already has that buff in any form, Bonnie will try to attack instead.


Buffing ATK

(Bonnie puffs their cheeks!)


Buffing DEF

(Bonnie yells encouragement everyone's way!)


Buffing ATKSPD

(Bonnie runs in a circle!)


Bonnie will try to attack if a 3, 4, or 5 is rolled, or if they couldn't buff

If Memory of Training is equipped, Bonnie will never miss.

If fighting against the King, Bonnie hits 70% of the time. Otherwise, they have a 50/50 chance of missing.


Without Bonnie's friendquest completed this loop

(Bonnie whacks the enemy!)


With Bonnie's friendquest completed this loop

(Bonnie whacks the enemy really hard!)


If they missed

(Bonnie missed...)


Bonnie deals the final hit

Source: Common Events 308 (FinalHit), 309 (FinalHit2); Skill 058 ([Bonnie's final blow])

Bonnie can deal the final hit on an enemy (except the tutorial Sadnesses, Bourdon and its hands, or the King) if it has less than 5% HP left after a party member takes their turn, killing them in one hit. The first time this happens, this text plays.

Bonnie (no!2) AW!!! We almost got it!!!

Mirabelle (anxious2) It's very weak... A slightly stronger blow would've done it!

Isabeau (fufufu1) Well, if it's weak...

Isabeau (brag1) Wanna deal the final blow, Bonnie?

Bonnie (wow1) CAN I?!??

Odile (dotdotdot2) It IS pretty weak, so...

Odile (lol3) Sure. Knock yourself out.

Bonnie (stop!2) O-OKAY!!!! HERE I GO!!!!!



(Bonnie deals the final blow!!!)


Bonnie (UWAH1) YEAAAAAAAAAAAAAAAAAAAH!!!!!!!


(You clap.)


Isabeau (hahaha2) Great job, Bonbon!

Odile (lol2) You did good.

Mirabelle (excited1) What would we have done without you!

Bonnie (hmf2) HMF!!!!!!!!!!!


[If an enemy is close to death, your little kiddo might deal the final blow!]

Mirabelle revives someone

Source: Common Events 305 (BaseTroop1), 306 (BaseTroop2), 307 (BaseTroop3); Troop54 (BASE TROOP); Skill 158 ([MEMORY_Revive])

With Memory of Buttkicking equipped, Mirabelle can revive someone as soon as they're knocked out. This is supposed to be able to happen once per battle, but in practice it seems to only happen once per save.

Reviving Siffrin

Mirabelle (fight4) Siffrin...!!!


Reviving Isabeau

Mirabelle (fight2) Isabeau, this is NOT the time!!!


Reviving Odile

Mirabelle (fight4) Odile, wake up...!!!

(Mirabelle brings her hands to the sky!)
