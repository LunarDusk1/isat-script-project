Drawn Card

For ACTs 2, 3, and 4, there are four possible versions of the Drawn Card that can be obtained at random. Only ACT 5 has a set card. Each card has a different description when interacted with.

The Drawn Card can only be interacted with in ACT 6 if it was not used in ACT 5, since Siffrin will rip it apart then.
